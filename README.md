## Shopping Cart Counter

A simple and interactive Shopping Cart Counter web app built using HTML, CSS, and JavaScript. Users can add items to the cart and see the cart count update instantly.

 ## Features

 * Add products to cart

 * Live cart item counter

* Modern gradient UI design

 * Hover animations on products and buttons

 * Responsive centered layout

## Demo

Click Add to Cart button on any product to increase the cart count.

## Tech Stack

HTML5 — Structure

CSS3 — Styling & Animations

JavaScript — Functionalit
